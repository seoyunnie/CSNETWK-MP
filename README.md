# PokeProtocol

## Setup

> [!IMPORTANT]
> Apache Maven is required to install the dependencies and easily run the application.

Install the dependencies:

```shell
mvn install
```

Compile to bytecode:

```shell
mvn compile
```

## Launching

```shell
mvn exec:java
```
